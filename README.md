# To--Do--List
An ultra-modern, fully interactive To-Do List web application built entirely in a single HTML file (with embedded CSS and JavaScript).  This project combines beautiful UI animations with powerful task management features like deadline tracking, notifications, and local storage support. 
#Features:
* Real-time clock display
* Add tasks with deadline (date & time)
* Mark tasks as completed
* Animated background image slider
* Glassmorphism UI design with smooth animations
* Task statistics (Total / Completed / Missed)
* Data persistence using localStorage
* Responsive and interactive design
